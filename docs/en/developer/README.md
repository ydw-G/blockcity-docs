# 开发者入驻

## 注册开发者账号

访问[公信宝开发者中心](https://developer.gxb.io/)注册登录账号。

![img](/img/d-1-1.png)

## 进入用户中心

登录开发者中心后，点击导航右上角用户信息，选择进入用户中心。

![img](/img/d-1-2.png)

## 账号认证

认证企业或个人信息（推荐认证企业信息，可获得更多权益），以便使用开放平台服务。

![img](/img/d-1-3.png)

![img](/img/d-1-4.png)

个人信息：

![img](/img/d-1-5.png)

## 绑定布洛克城账号

回到用户中心，点击绑定布洛克城账号，如未安装过布洛克城Dapp，可访问[布洛克城官网](https://blockcity.gxb.io/download/)下载并注册账号。
